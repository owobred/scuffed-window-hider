## Compiling / running
1. Install rust
2. Install the nightly toolchain
3. Install the `i686-pc-windows-msvc` target
4. Install [`cargo-make`](https://github.com/sagiegurari/cargo-make#installation)
5. Run `cargo make build`, and then `cargo run`
